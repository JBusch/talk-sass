Was ist Sass? / Begriffsfindung

In Ruby geschrieben und compiliert via Ruby

LibSass ist ein Sass Port in C/C++
(Momentan kein compass Support)

Es gibt zwei verschiedenen Sass Syntaxen

Sass 

#main
  color: blue
  font-size: 0.3em

  a
    font:
      weight: bold
      family: serif
    &:hover
      background-color: #eee
und SCSS 

#main {
  color: blue;
  font-size: 0.3em;

  a {
    font: {
      weight: bold;
      family: serif;
    }
    &:hover {
      background-color: #eee;
    }
  }
}

Installation

gem install ruby
gem install sass

CMD Befehle

sass --watch
sass --watch input.sass:output.css
sass --watch input-dir:output-dir
