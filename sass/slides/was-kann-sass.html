Was kann Sass?

Wie erleichtert Sass unser leben?

Variablen in CSS nutzen!

Sass Variablen werden definiert:
$name: wert;

Variablen in CSS nutzen!

// Fonts
$font-family-screen: Arial, Helvetica, sans-serif;
$font-family-print: "Times New Roman", Times, serif;

// Colors
$gray: #ccc;
$green: #33cc33;

$primary-color: $gray;
$secondary-color: $green;

// Misc
$margin: 0.2vw;

p {
	font-family: $font-family-screen;
	color: $gray;
	margin-bottom: $margin;
}
kompiliert zu 

p {
	font-family: Arial, Helvetica, sans-serif;
	color: #ccc;
	margin-bottom: 0.2vw;
}

Verschachtelung!

element { noch-ein-element { ...{} } }

Verschachtelung!

.news-list-container {
  margin: 0;
  padding: 0;

  .news-list-item {
    border: 1px solid $gray;
    padding: $padding;
      
      a {
        color: $primary-color;
        &:hover {
            color: $secondary-color;
            text-decoration: underline;
        }
      }
    }
}
kompiliert zu 

.news-list-container {
	margin: 0;
	padding: 0;
}

.news-list-container .news-list-item {
	border: 1px solid #ccc;
	padding: 0.5vw;
}

.news-list-container .news-list-item a {
	color: #ccc;
}

.news-list-container .news-list-item a:hover {
	color: #33cc33;
	text-decoration: underline;
}
